Google Sheets Integration

Deploy your Google Apps Script as a Web App and paste the URL here. Each confirmed PDIO check will be saved automatically to your Google Sheet.

Google Apps Script Code

Go to script.google.com → New project → paste the code → Deploy as Web App (Execute as: Me, Who has access: Anyone). Copy the deployment URL and paste it above. 
function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sh = ss.getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  if (sh.getLastRow() === 0) {
    sh.appendRow(['Timestamp','PDIO Number',
      'Order Type','PDIO Qty','Checked By', 'Employee ID',
      'Total Items','Item Details']);
  }
  sh.appendRow([
    data.datetime,
    data.pdioNumber,
    data.orderType,
    data.pdioQty,
    data.checkedBy,
    data.empId,
    data.totalItems,
    JSON.stringify(data.items)
  ]);
  return ContentService
    .createTextResponse('OK')
    .setMimeType(ContentService.MimeType.TEXT);
}

Scanner Tips

Hardware barcode/QR scanners emulate a keyboard — they type the scanned value and send Enter automatically.

• The scan field is auto-focused when the Scan tab opens.
• After a scan, the PDIO loads immediately — no button press needed.
• If the field loses focus (e.g. you clicked elsewhere), click the scan field or press Tab to refocus before scanning.
• You can also type a PDIO number manually and press Enter or tap Load PDIO.
